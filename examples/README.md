## Contributing Examples

Have a great real-world example? Contribute it!

### Guidelines

1. **Anonymize**: Remove company-specific information
2. **Complete**: Show full output, not snippets
3. **Realistic**: Based on actual usage, not hypothetical
4. **Document**: Add explanation of context and approach
5. **Format**: Follow existing example structure

### Submission

1. Create example file in appropriate directory
2. Update this README with summary
3. Submit pull request with description

## Example Quality Checklist

Good examples should:
- [ ] Be based on real scenarios
- [ ] Show complete output, not partial
- [ ] Demonstrate skill features effectively
- [ ] Include context and rationale
- [ ] Follow proper formatting
- [ ] Be anonymized (no sensitive data)
- [ ] Help users understand expected output
- [ ] Show best practices in action
